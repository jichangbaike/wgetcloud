## 常见问题

### 1）为什么 WgetCloud 价格偏贵？

整体产品更偏向“稳定型专线服务”，定价通常会包含更高的线路成本与运维成本，适合把它当作长期工具使用的用户。

### 2）我该选哪个等级？

- 轻度使用：基础专线即可
- 主力长期使用：优质专线更均衡
- 对峰值带宽、节点数量与高可用更敏感：精品专线更合适

### 3）是否适合多人/团队共享？

页面说明一般为**仅限账户本人使用，不支持团队使用**（以官网条款为准）。

### 4）是否承诺某些地区可用？

页面一般会出现类似“不承诺某地区用户可用性”的描述；建议购买前确认你所在地网络环境与服务可达性。
